# Makeki Bakery
A fully responsive 8-page website for Makeki Bakery, Nairobi.
Built with HTML5, CSS3, JavaScript ES6+ and Bootstrap 5.

## Live Site
https://MikeNgugi-hub.github.io/Makeki_Bakery

## Pages
- Home · About · Menu · Gallery · Blog · Contact · FAQ · Reviews

## JavaScript Features
1. Contact form validation with inline error messages
2. Dark/light mode toggle (saved in localStorage)
3. Gallery filter + menu live search + modal lightbox
